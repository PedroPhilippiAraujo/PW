    Projeto Tutor de IA PT-BR

    O projeto consiste em um programa que leia textos em português e faça correções (gramaticais, ortográficas e sintaxe) utilizando IA. Além disso o usuário pode utilizar outras vantagens como aprimorar o texto com intuito dele ser mais formal e profissional ou resumi-lo em um número de linhas.

    O programa tem dois tipos de usuários, o comum, que utiliza o site normalmente; e o super usuário, que tem acesso aos dados dos usuários comuns. Nesse sentido a primeira aba do site é o cadastro, para cria-lo o usuário precisa digitar seus dados (usuario e senha). Dentro do site o usuário comum pode salvar seus textos.

    Recursos usados para o Projeto

    O recurso que utilizado no projeto é a API do Hugging Face, equipado com o modelo Llama-3.1-8B da Meta. Pois o mesmo possui acesso gratuito à estudantes e suporte a língua Português.
